step 1 : install FTP server
. install FTP server ( sudo apt install vsftpd)
. if it was installed then 
. start service with the command ( sudo systemctl status vsftpd)
. check if the vsftpd service is active or not ( sudo systemctl status vsftpd)
. if your vsftpd is not active then type ( sudo systemctl enable--now vsftpd )
